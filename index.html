How to generate a sheet with ChatGPT

ChatGPT won't output an exact pixel size, and that's fine — this tool auto-detects the size and splits the width into 6 equal frames. What matters: exactly 6 frames, a truly transparent background, and the character staying still between frames.

Copy this prompt and replace the [brackets]: 
Create a 6-frame pixel art sprite sheet of [CHARACTER, e.g. a shiba inu] for a social media game called Necho.

[CHARACTER] is [SHAPE + PERSONALITY, e.g. small, round, chubby, and unbothered], in a neutral resting pose.
Key features: [2-4 DISTINCT FEATURES, e.g. big dark eyes, brown muzzle, cream belly, small rounded paws].

Show a subtle idle loop. Keep the character IDENTICAL in every frame - same size, same position, same colors - and move only tiny details by 1-2 pixels:
- Frame 1: neutral resting pose
- Frame 2: body rises slightly, as if breathing in
- Frame 3: back to neutral
- Frame 4: [SMALL ACCENT MOTION, e.g. ears flop down / eyes blink / tail flicks]
- Frame 5: [the accent continues or peaks]
- Frame 6: back to neutral, so the loop returns smoothly to Frame 1

Style: thick dark pixel outline, soft pixel shading, big expressive eyes, cute Japanese game-mascot look, chunky low-resolution pixels.

Layout & technical requirements (critical for the animation to work):
- Exactly 6 frames, arranged left to right in ONE single horizontal row.
- All 6 frames the SAME width, evenly spaced, with equal gaps between them and equal margins on the left and right edges.
- The character is the same size and centered the same way in every frame, resting on a shared baseline, so it does not drift or jump between frames.
- Fully transparent background (real alpha) - no white fill, no color fill, no checkerboard, no border, no boxes or grid lines around the frames.
- No ground shadow, no background scenery, no text or frame numbers, no UI, no extra characters - only the character floating on transparent space.
- Wide landscape strip, about 6 times wider than one frame.
- Export as a transparent PNG.

Before uploading: count 6 frames · confirm the background is transparent (not white) · check the character doesn't drift. If the background comes out white, reply to ChatGPT with "make the background fully transparent (alpha), remove any white" and regenerate.
